## Setup

```sh
nvm use
npm install
npm run dev
```

By default, the following services will be available:

- React App at <http://localhost:3005/>
- GraphQL server at <http://localhost:3000/graphiql>
- Postgres at <postgres://user:pass@localhost:5432/db>
